Tarefa sobre Formulário

Maria Eduarda Nunes

Questionário

Perguntas e Resposta

A. Na tag < input > com atributo type="text", para que serve o atributo "placeholder"?

Resposta: Serve para fornecer um texto de exemplo dentro do campo de entrada de texto.

B. Na tag < input > com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?

Resposta: Qual opção de "rádio" deve ser selecionada por padrão quando a pagina for carregada.

C. Na tag < input > para que serve o atributo "value"?

Resposta: Para especificar o valor associado a um campo de entrada.

D. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

Resposta: O atributo "name"

E. Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?

Resposta: Define um rotulo para varios elementos de formulario. O atributo "for" especifica qual elemento de formulario esta associado ao rotulo.

F.Na tag < option > para que serve o atributo "selected"?

Resposta: Qual opção deve ser selecionada por padrão quando a pagina for carregada.

G.Nas tags de formulario para que serve o atributo "required"?

Resposta: Especifica que um campo de entrada é obrigatório.

H.Para que serve a tag < textarea >?

Resposta: Define um campo de entrada de várias linhas (área de texto).

I. Na tag < form > para que serve o atributo "action"?

Resposta: Define a ação a ser executada quando a usuário clica no botão enviar.

J. Na tag < form > para que serve o atributo "method"?

Resposta: Especifica o metodo Http a ser usado ao enviar os dados do formulário.

K.Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag < form > ?

Resposta: Method="post"

L.Qual valor deve ser informado no atributo "method" na tag < form > em um formulário que será usado para fazer upload de arquivos?

Resposta: Method="get"

M.O que as tags <fieldset> e <legend> produzem?

FIELDSET=Usado para agrupar dados relacionados em um formulário.

LEGEND=Define uma legenda para o fieldset-elemento.
